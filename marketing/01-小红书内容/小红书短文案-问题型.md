## 📊 短文案特点

| 特点 | 说明 |
|------|------|
| **字数** | 标题15-20字，正文30-50字 |
| **结构** | 问题/痛点 + 提问互动 |
| **目的** | 引发共鸣，引导评论 |
| **风格** | 口语化，像朋友聊天 |
| **标签** | 4-6个，精准定位 |

---

## 🎯 发布策略

1. **先发问题型文案**引发讨论
2. **评论区互动**收集真实需求
3. **再发解决方案**（产品介绍）承接流量
4. **形成闭环**：问题→讨论→方案→转化
