## 🎨 设计规范

### 色彩系统

**主色调**：
- 粉色：#FF9A9E
- 橙色：#FECFEF
- 渐变：linear-gradient(135deg, #FF9A9E 0%, #FECFEF 100%)

**辅助色**：
- 文字主色：#333333
- 文字次色：#666666
- 背景色：#FFF5F5
- 分割线：#EEEEEE

### 字体规范

**标题**：
- 中文：PingFang SC / Microsoft YaHei
- 英文：SF Pro Display / Helvetica
- 字号：18-24px
- 字重：Bold

**正文**：
- 字号：14-16px
- 字重：Regular
- 行高：1.5-1.8
