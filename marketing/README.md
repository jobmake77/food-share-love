## 📊 数据追踪

每次发布后，记录以下数据：

**小红书数据**：
- 发布时间
- 曝光量
- 阅读量
- 点赞数
- 收藏数
- 评论数
- 转发数
- 涨粉数

**小程序数据**：
- 当日新增用户
- 当日活跃用户
- 来源渠道（从哪个帖子来的）

使用 `05-数据复盘/数据记录模板.md` 进行记录。

## 🔄 迭代优化

基于数据，每两周进行一次复盘：

1. **分析爆款**：哪些帖子数据好？为什么？
2. **总结规律**：标题、封面、内容的规律
3. **优化方向**：下次如何改进？
4. **更新素材库**：把好的内容沉淀到素材库
